## Runtime entry points

Use `ProxyWhirl` for synchronous scripts and `AsyncProxyWhirl` for async applications. The REST API is mounted under `/api/*` when running `proxywhirl.api:app`, and CLI commands are available through `uv run proxywhirl`.

## Authentication

Production API deployments should set `PROXYWHIRL_REQUIRE_AUTH=true` and provide `PROXYWHIRL_API_KEY`. Health and readiness endpoints remain public for probes; metrics and stats exposure should be restricted by deployment policy when operational metadata is sensitive.
