The OpenAPI endpoint pages are generated from `proxywhirl.api.app.openapi()` and grouped by operational area.

| Group | Covers |
| ----- | ------ |
| [Configuration](/docs/api/openapi/configuration) | Runtime configuration management |
| [Monitoring](/docs/api/openapi/monitoring) | Health checks, status, and metrics endpoints |
| [Pool Management](/docs/api/openapi/pool-management) | Manage proxy pool (add, remove, list proxies) |
| [Proxied Requests](/docs/api/openapi/proxied-requests) | Make HTTP requests through rotating proxies |
| [Retry & Failover](/docs/api/openapi/retry-&-failover) | Retry policies, circuit breakers, and failover metrics |

For a compact route inventory, see the [REST/OpenAPI Reference](/docs/generated/rest-api).
