# TUI

ProxyWhirl’s TUI is the interactive terminal surface for local inspection and operations.

Use the TUI when you need a human-operated dashboard for proxy inventory, source status, health, or exports.

```bash
uv run proxywhirl tui
```

The TUI is complementary to the CLI and REST API. Use the CLI for automation, REST for service integrations, and TUI for live local inspection.
