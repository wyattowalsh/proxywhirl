### Cache Formats

<Tabs items={['In-Memory', 'JSON']}>
  <Tab value="In-Memory">
    **Lightning Fast Performance**
    
    Store proxy data directly in memory for maximum speed and minimal latency. Perfect for high-throughput applications where performance is critical.
    
    ```python
    from proxywhirl import proxywhirl
    
    proxy_whirl = proxywhirl(cache_type="memory")
    ```
  </Tab>
  <Tab value="JSON">
    **Human-Readable Storage**
    
    Persist proxy configurations and state in JSON format for easy debugging, backup, and manual editing when needed.
    
    ```python
    proxy_whirl = proxywhirl(
        cache_type="json",
        cache_path="proxies.json"
    )
    ```
  </Tab>
</Tabs>

### Intelligent Rotation Strategies

<Tabs items={['Round Robin', 'Random', 'Weighted', 'Health-Based']}>
  <Tab value="Round Robin">
    **Sequential Distribution**
    
    Distribute requests evenly across all available proxies in a predictable order.
    
    ```python
    proxy_whirl = proxywhirl(rotation_strategy="round_robin")
    
    # Each request uses the next proxy in sequence
    for i in range(5):
        proxy = await proxy_whirl.get_proxy()
        print(f"Request {i}: {proxy.host}:{proxy.port}")
    ```
  </Tab>
  <Tab value="Random">
    **Unpredictable Selection**
    
    Randomly select proxies to avoid detection patterns and distribute load unpredictably.
    
    ```python
    proxy_whirl = proxywhirl(rotation_strategy="random")
    
    # Random proxy selection for each request
    proxy = await proxy_whirl.get_proxy()
    ```
  </Tab>
  <Tab value="Weighted">
    **Performance-Based Priority**
    
    Prioritize faster, more reliable proxies while still utilizing the entire pool.
    
  ```python
  # Weighted prefers faster proxies (lower response_time)
  proxy_whirl = proxywhirl(rotation_strategy="weighted")
  ```
  </Tab>
  <Tab value="Health-Based">
    **Adaptive Selection**
    
    Automatically avoid unhealthy proxies and favor high-performing ones.
    
    ```python
    proxy_whirl = proxywhirl(
        rotation_strategy="health_based",
        health_check_interval=30
    )
    ```
  </Tab>
</Tabs>
