# Placeholder for generated proxy lists

This directory contains automatically generated proxy lists from 64+ free sources.

Files are updated every 6 hours via GitHub Actions.

## Available Files

- `http.txt` - HTTP/HTTPS proxies (one per line)
- `socks4.txt` - SOCKS4 proxies (one per line)
- `socks5.txt` - SOCKS5 proxies (one per line)
- `all.txt` - All proxies combined with headers
- `proxies.json` - JSON format with metadata
- `metadata.json` - Generation stats and timestamp

## Usage

Download directly from the docs site or use the ProxyWhirl library to fetch them programmatically.
